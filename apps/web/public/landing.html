Setup instructions

Directely is built as a local-first application: a web studio talks to a Python API and background workers. The smoothest path on Windows is the launcher script after Docker is running.

Prerequisites

Install the following on the machine that will run the studio:

  • Windows 10/11 (or follow INSTALLATION.md for Linux server installs)
  • Docker Desktop — Postgres, Redis, and optional MinIO run in containers (docker compose).
  • Node.js 18+ — for the web UI and Electron packaging.
  • Python 3.11+ — for the API and Celery workers.
  • FFmpeg on PATH (or set FFMPEG_BIN in your environment) for compile / export jobs.

1. Get the software

After purchase you receive a download or installer (e.g. Directely Setup from the Electron build) plus license terms. Keep your .env out of any bundle you share publicly.

2. Configure environment

Copy .env.example to .env at the project root. Set database URL, Redis URL, and any AI or stock API keys your workflow needs. Default storage is the filesystem under your data directory—see LOCAL_STORAGE_ROOT in the example file.

3. Start Docker services

From the repository root:

docker compose up -d

Then run database migrations (from apps/api with your venv active): 

alembic upgrade head

4. Run API, worker, and beat

You need three processes: the FastAPI server, a Celery worker (queues text, media, compile), and Celery beat for scheduled tasks like stale-job reaping. 

# Terminal A (apps/api)
python -m director_api

# Terminal B
celery -A director_api.tasks.celery_app worker -Q text,media,compile -l info

# Terminal C
celery -A director_api.tasks.celery_app beat -l info

5. Windows one-shot launch

From the repo root, after Docker is up and .env exists, you can use the bundled launcher (starts stack pieces and opens the browser—see script for exact behavior): 

.\Launch.ps1

Alternatively use Launch.cmd or the restart-local.ps1 helper if your team provides it.

6. Open the studio

Development UI: from apps/web run npm install then npm run dev and open the URL Vite prints (API is usually proxied to /v1). Production: use the built static files or the installed Electron app per your distribution README.

Stuck?

Email support@directely.com with your OS version and where the install failed (Docker, migrations, worker, or browser).
